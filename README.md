# FitLife

FitLife is a React fitness website with pages for home, about, classes, trainers, pricing, shop, blog, contact, and membership signup.

## Scripts

- `npm start` runs the development server.
- `npm run build` creates a production build.

## Project Setup

This project uses `react-scripts`, with:

- `src/index.js` as the app entry point
- `public/index.html` as the HTML template

Unused Vite starter files and duplicate components were removed so the structure matches the active app setup.
